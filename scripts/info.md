# Scripts Directory

## Overview
This directory contains all the scripts required to run the various stages of the StrainCascade pipeline.

## Structure
- 

## Script list
- **main_script.sh**: The main script that orchestrates the entire pipeline.
- **data_preprocessing.sh**: Prepares raw data for analysis.
- **variant_calling.sh**: Performs variant calling on processed data.
- **qc_analysis.sh**: Conducts quality control checks on the results.

## Usage
- Make sure all scripts are executable (`chmod +x script_name.sh`).
